## 効果的な質問のコツ

### ❌ 避けるべき質問

```
Next.jsについて教えて
```
→ 曖昧すぎて、どの情報が必要か不明確

### ✅ 良い質問

```
Next.js 15の公式ドキュメントから、App Routerでの
動的ルーティングの実装方法を調べてください
```
→ 具体的で、情報源も明確

### 質問のコツ

1. **具体的に**
   - 何について知りたいのか明確に
   - バージョンを指定する

2. **目的を伝える**
   - 「〜を実装したい」
   - 「〜を理解したい」

3. **情報源を指定**
   - 「公式ドキュメントから」
   - 「最新のベストプラクティスを」

4. **段階的に深掘り**
   - まず概要を理解
   - 次に詳細を調査
   - 最後に実装方法を確認
