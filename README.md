## 🤖 Claude Codeとは？

**AnthropicによるAIペアプログラミングツール**

- ターミナルで動作するCLIツール
- Claude 3.5 Sonnetの強力なAI能力を活用
- プロジェクト全体のコンテキストを理解
- 自律的なタスク実行が可能

### できること

| 機能                       | 説明                         |
| -------------------------- | ---------------------------- |
| 🎨 **コード生成**           | 機能説明からコードを自動生成 |
| 🐛 **デバッグ**             | バグの発見と修正提案         |
| 🗺️ **コードナビゲーション** | 大規模コードベースの理解     |
| ⚡ **自動化**               | 繰り返し作業の効率化         |
